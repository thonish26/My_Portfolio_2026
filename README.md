# 🌌 Developer Portfolio

Modern 3D portfolio website built with React and Three.js.

---

## ✨ Features
- 3D Galaxy Background
- Theme Toggle (Dark/Light)
- Animated Hero Section
- Responsive Design
- Lottie Animations

---

## 🛠 Tech Stack
React, Three.js, CSS, Lottie
