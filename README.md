# My Portfolio 2026 🚀

A modern responsive portfolio website built with React.js, CSS, Three.js animations and interactive UI components.

## 🌐 Live Demo

https://portfoilo2026-nu.vercel.app/

## 🛠️ Technologies Used

- React.js
- JavaScript
- CSS3
- Three.js
- React Three Fiber
- Lottie Animation
- Vercel Deployment

## ✨ Features

- Animated Galaxy Background
- Interactive Hero Section
- Responsive Design
- Dark / Light Theme
- Project Showcase
- Contact Section
